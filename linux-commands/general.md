# General Linux commands

time- runs a program and summarizes system resource usage
```bash
rajsekhar@thinkpad:~/notes/linux-commands$ time ls
general.md

real	0m0.003s
user	0m0.000s
sys	    0m0.003s
```

open Text Editor:
```bash
gnome-text-editor # Opens text editor with an empty file
gnome-text-editor notes/file.txt # Opens a file
```

systemctl suspend- suspends laptop (sleep)

open File Explorer:
```bash
nautilus # Opens home directory
nautilus . # Opens current directory
nautilus projects/pa # Opens specified directory
```
