# Linux tree command

Displays all contents of a folder in a tree like structure
```bash
tree
```

Flags:
```bash
-a : Display hidden files

-s : Display size of each file in bytes

-h : Display sizes in human-readable format

-L [level] : Max display depth of tree
    -L 1 : One level of files/folders

-D : Display time of last modification

-t : Sort by last modification time

--gitignore : Ignore contents mentioned in .gitignore

--du : Display total size of each folder

--sort=size : sorts by size
```
