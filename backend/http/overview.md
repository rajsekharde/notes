# HTTP overview

HTTP - A client-server protocol for fetching resources such as HTML documents. It's the foundation for any data exchange over the web.

Client and server communicate by exchanging invividual messages instead of streaming. It is initiated by a client by sending a message called 'request' to a server. The server processes the request and sends a message called 'response' to the client.

HTTP is an application-layer protocol that is sent over TCP, or over a TLS-encrypted TCP connection.
